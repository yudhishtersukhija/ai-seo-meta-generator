# AI SEO Meta Generator

A WordPress plugin that generates SEO titles and meta descriptions using AI.

## Features

* Custom SEO Meta Box
* AJAX-Based Generation
* Manual Editing Support
* WordPress Settings Page
* OpenAI Ready Architecture
* Secure Nonce Verification
* OOP Plugin Structure

## Installation

1. Upload plugin folder to `/wp-content/plugins/`
2. Activate plugin
3. Open WordPress Admin
4. Configure API settings
5. Start generating SEO metadata

## Technologies

* PHP
* WordPress Plugin API
* AJAX
* JavaScript
* OOP
* WordPress Settings API
